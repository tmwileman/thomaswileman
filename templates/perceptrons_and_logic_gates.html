Perceptrons as Logic Gates

Perceptrons are designed to mimic the decision processes that take place in the human brain. In Perceptron Primer we discussed the structure of a perceptron and how it works. An important component of that structure are the weights used to modify inputs and bias. These weights can be modified to create logical operators; simple models take in two or more inputs, perform a boolean operation (true or false), and return a binary result. Importantly, three simple logical operators, AND, OR, and NOT, have been found to manipulate data in much the same way our brains do. This means that by encoding perceptrons with simple logical operators and placing them in various configurations we can simulate how our brain's process information.

The natural language equivalent of the OR logic gate is "if A or B are True return True, else return False". For example, if A is True and B is False an AND operator will return True. Consider an example in which we would like to generate an OR operator for the same linear function used above, I1W1 + I2W2 + B = 0. Given the following inputs, [(0,0), (1,0), (0,1), (1,1)], we expect the OR operator to give the following results, [False, True, True, True]. We need to identify a combination of weights and bias that will provide the desired results.

An interesting way to think about this considering how we might make an OR operator out of an AND operator. This can be done by increasing the weights of each input to the inverse of the bias (e.g. each weight has a value of 2 and the bias has a value of -2). Similarly, we could decrease the value of the bias. Each of these methods has the effect of making any one of the inputs enough to counteract the bias, resulting in a value of 1.
